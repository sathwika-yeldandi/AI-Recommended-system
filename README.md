
# 🎬 AI Movie Recommendation System

A simple AI-based Movie Recommendation System developed using Python. The system recommends movies to users based on their preferred genre using rule-based recommendation logic.

---

## 📌 Project Overview

This project demonstrates the basic concept of an AI Recommendation System by matching user preferences with movie genres. It uses pattern matching instead of machine learning, making it an excellent beginner AI project.

---

## ✨ Features

- Takes user input (favorite movie genre)
- Searches a movie dataset
- Recommends movies based on user preference
- Beginner-friendly implementation
- Easy to customize with more movies and genres

## 📊 Dataset

The project uses a CSV file (`movies.csv`) containing:

- Movie Name
- Genre

Example:

| Movie | Genre |
|--------|--------|
| Avengers | Action |
| Titanic | Romance |
| Interstellar | Sci-Fi |
| Conjuring | Horror |

---

## ⚙️ How It Works

1. User enters a movie genre.
2. The program reads the movie dataset.
3. It compares the user's preference with available genres.
4. Matching movies are recommended.
5. Results are displayed on the screen.
